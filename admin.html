Instructions

How to Use the Admin Panel:

  1. Create a Post: Fill in all required fields marked with *
  2. Markdown Support: Write your content using Markdown syntax
  3. Save Posts: Click "Save Post" to download the updated data file
  4. Upload Data: After saving, copy the downloaded JSON to data/posts.json
  5. Edit Posts: Click "Edit" on any post to load it into the form
  6. Featured Posts: Check the "Featured Post" box to highlight on homepage
  7. Images: Use direct image URLs (from GitHub, Imgur, etc.)

Data Storage:

Posts are stored in data/posts.json. Since GitHub Pages is static, you need to manually update this file after creating/editing posts.

Quick Workflow:
1. Create/edit post → 2. Click "Save Post" → 3. Copy JSON content → 4. Paste into data/posts.json → 5. Commit to GitHub
